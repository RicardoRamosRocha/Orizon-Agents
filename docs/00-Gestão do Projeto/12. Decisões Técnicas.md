# Decisões Técnicas

## Objetivo

Este documento registra as principais decisões técnicas tomadas durante o desenvolvimento do Orizon Agents.

Seu objetivo é documentar os motivos de cada decisão e facilitar futuras manutenções.

---

# Decisões Atuais

## Arquitetura

- Utilizar Clean Architecture.
- Separação em Domain, Application, Infrastructure e Presentation.

---

## Tecnologia

- ASP.NET Core 8
- Entity Framework Core
- PostgreSQL
- Redis
- Docker

---

## Inteligência Artificial

- Arquitetura preparada para múltiplos provedores.
- Integração inicial com OpenAI.
- Expansão futura para Gemini, Claude e Azure OpenAI.

---

## WhatsApp

- Utilizar a WhatsApp Cloud API oficial da Meta.

---

## Banco de Dados

- PostgreSQL como banco principal.
- Migrations utilizando Entity Framework Core.

---

## Interface

- ASP.NET Core MVC.
- Interface responsiva.
- Design System próprio da Orizon Platform.

---

## Atualizações

Toda decisão arquitetural importante deverá ser registrada neste documento.
