# Sprint 5 - WhatsApp Cloud API

Fundação técnica para integração oficial com a Meta WhatsApp Cloud API.

## Configuração mínima em desenvolvimento

1. Criar um aplicativo no Meta for Developers.
2. Associar ou criar uma WhatsApp Business Account.
3. Obter o WABA ID e o Phone Number ID no painel da Meta.
4. Configurar segredos por User Secrets ou variáveis de ambiente:
   - `WhatsApp__GraphApiVersion`
   - `WhatsApp__AppSecret`
   - `WhatsApp__VerifyToken`
5. Configurar o webhook HTTPS público apontando para `/api/webhooks/whatsapp`.
6. Cadastrar o mesmo Verify Token configurado na aplicação.
7. Assinar o campo `messages` no produto WhatsApp.

Tokens de acesso dos tenants são armazenados criptografados pela aplicação e não devem ser registrados em documentação, logs ou arquivos versionados.
