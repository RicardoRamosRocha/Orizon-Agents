# Sprint 1

## Objetivo

Desenvolver o primeiro MVP funcional do Orizon Agents.

---

# Funcionalidades

- Cadastro de organizações.
- Cadastro de usuários.
- Login.
- Gestão de agentes.
- Configuração básica dos agentes.
- Integração com um provedor de IA.
- Integração com WhatsApp Cloud API.
- Recebimento e envio de mensagens.
- Histórico de conversas.
- Dashboard inicial.

---

# Critério de Conclusão

Ao final desta Sprint será possível criar um agente, conectá-lo ao WhatsApp e responder mensagens utilizando Inteligência Artificial.
