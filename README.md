# 📖 Sobre o Projeto

O **Orizon Agents** é uma plataforma SaaS desenvolvida para permitir que empresas criem, configurem e gerenciem agentes de Inteligência Artificial integrados ao **WhatsApp**.

O objetivo é fornecer uma infraestrutura moderna, escalável e segura para automatizar atendimento, vendas, suporte e processos internos utilizando múltiplos provedores de IA.

---

# ✨ Principais Recursos

- 🤖 Criação e gerenciamento de Agentes de IA
- 💬 Integração com WhatsApp Cloud API
- 🧠 Suporte a múltiplos provedores de IA
- 📚 Base de Conhecimento
- 🔎 RAG (Retrieval-Augmented Generation)
- 🛠️ Ferramentas (Tools)
- 👨‍💼 Atendimento Humano
- 📊 Dashboard e Métricas
- 🏢 Arquitetura Multi-Tenant
- 💳 Planos e Assinaturas
