# 📖 Sobre o Projeto

O **Orizon Agents** é uma plataforma SaaS desenvolvida para permitir que empresas criem, configurem e gerenciem agentes de Inteligência Artificial integrados ao WhatsApp.

A plataforma foi projetada para oferecer uma infraestrutura moderna, escalável e segura, permitindo automatizar atendimento, vendas, suporte e processos de negócio utilizando múltiplos provedores de IA.

---

# ✨ Principais Recursos

- 🤖 Criação de Agentes de IA
- 💬 Integração com WhatsApp Cloud API
- 🧠 Múltiplos Provedores de IA
- 📚 Base de Conhecimento
- 🔍 RAG (Retrieval-Augmented Generation)
- 🛠 Ferramentas (Tools)
- 👨‍💼 Atendimento Humano
- 📊 Dashboard e Métricas
- 🏢 Arquitetura Multi-Tenant
- 💳 Gestão de Planos e Assinaturas
