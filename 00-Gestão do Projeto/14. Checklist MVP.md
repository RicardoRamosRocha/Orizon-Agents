# Checklist MVP

## Objetivo

Este checklist define os requisitos mínimos para que a primeira versão do Orizon Agents possa ser apresentada a clientes.

---

# Infraestrutura

- [ ] Repositório organizado.
- [ ] Banco PostgreSQL configurado.
- [ ] Migrations funcionando.
- [ ] Ambiente de desenvolvimento configurado.

---

# Segurança

- [ ] Login funcionando.
- [ ] Controle de permissões.
- [ ] Multi-Tenant validado.

---

# Funcionalidades

- [ ] Cadastro de organizações.
- [ ] Cadastro de usuários.
- [ ] Gestão de agentes.
- [ ] Integração com OpenAI.
- [ ] Integração com WhatsApp Cloud API.
- [ ] Histórico de conversas.

---

# Interface

- [ ] Dashboard.
- [ ] Layout administrativo.
- [ ] Responsividade.
- [ ] Design System.

---

# Qualidade

- [ ] Build sem erros.
- [ ] Testes principais executados.
- [ ] Documentação atualizada.

---

# Validação

O MVP será considerado concluído quando um cliente conseguir criar um agente, conectá-lo ao WhatsApp e realizar um atendimento utilizando Inteligência Artificial.
