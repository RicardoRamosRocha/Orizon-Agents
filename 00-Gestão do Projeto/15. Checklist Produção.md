# Checklist Produção

## Objetivo

Este documento reúne os itens que devem ser verificados antes da publicação do Orizon Agents em ambiente de produção.

Nenhuma nova versão deverá ser publicada sem que este checklist seja revisado.

---

# Infraestrutura

- [ ] Ambiente de produção configurado.
- [ ] Banco de dados em produção.
- [ ] Backup automático configurado.
- [ ] Certificado HTTPS válido.
- [ ] Variáveis de ambiente configuradas.

---

# Segurança

- [ ] Segredos protegidos.
- [ ] Autenticação funcionando.
- [ ] Controle de permissões validado.
- [ ] Multi-Tenant testado.

---

# Aplicação

- [ ] Build sem erros.
- [ ] Migrations executadas.
- [ ] Logs funcionando.
- [ ] Health Checks configurados.
- [ ] Monitoramento ativo.

---

# Integrações

- [ ] WhatsApp Cloud API validada.
- [ ] Provedor de IA funcionando.
- [ ] Webhooks configurados.

---

# Qualidade

- [ ] Testes executados.
- [ ] Documentação atualizada.
- [ ] Revisão final concluída.

---

# Aprovação

A publicação somente poderá ocorrer após validação técnica e aprovação da equipe responsável.
